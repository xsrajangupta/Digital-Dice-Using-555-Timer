 Digital Dice Using 555 Timer IC

This project demonstrates a Digital Dice built using the NE555 Timer IC and a 7-Segment Display. The circuit was designed and tested in Proteus to simulate the working of an electronic dice.
A Verilog simulation was also performed to study and verify the digital logic behavior.

Objective
Design an electronic digital dice using the 555 Timer IC.
Display random numbers on a 7-segment display.
Simulate the complete circuit in Proteus.
Verify the logic through Verilog simulation.

Components Used
- NE555 Timer IC
- 7-Segment Display
- Resistors
- Capacitors
- Push Button
- Power Supply
- Proteus Design Suite

 Working Principle

When the push button is pressed, the 555 Timer generates pulses that rapidly change the displayed number. After the button is released, the display stops at a random number between 1 and 6, behaving like a digital dice.

Software Used

- Proteus Design Suite
- Verilog HDL (Simulation)
